# Basic Button

사용자 행동을 유도하는 용도로 쓰이는 라운드 형태를 접목한 사각형 형태의 버튼  
버튼의 사용처에 따라 크기(Large/Medium/Small과 Tiny)로 사용  
너비는 배치 상황과 환경에 따라 가변적으로 사용  
의미와 우선 순위에 따라 색상(Primary Color의 변형 / Gray)으로 구분
